下载地址:

https://github.com/xmrig/xmrig/releases

LINUX版本在本地编译，编译方法如下:

https://xmrig.com/docs/miner/ubuntu-build 为了通用性， 按照Advanced 方法编译就行。



生成配置文件:

https://xmrig.com/wizard#result

Misc选择 1%, 如果不选会或写0会默认为 5%。

生成结果像这样:

win

```powershell
xmrig.exe
 --donate-level 1 -o us-west.minexmr.com:443 -u 
45jcC272Sv9HuZu5D1giXpV8B4fkuN5muiU5UGx2DJqKTsVykfsJYraZ5FqBSo9Smc777NMxpPpmtNz5i3un9vozDs5gScR
 -k --tls --rig-id MY_ADDRESS
```



linux

```bash
./xmrig
 --donate-level 1 -o us-west.minexmr.com:443 -u 
45jcC272Sv9HuZu5D1giXpV8B4fkuN5muiU5UGx2DJqKTsVykfsJYraZ5FqBSo9Smc777NMxpPpmtNz5i3un9vozDs5gScR
 -k --tls --rig-id MY_ADDRESS
```



注意：如果要后台运行， 记得加上 -B 参数上去，如:

```bash
./xmrig
 --donate-level 1 -o us-west.minexmr.com:443 -u 
45jcC272Sv9HuZu5D1giXpV8B4fkuN5muiU5UGx2DJqKTsVykfsJYraZ5FqBSo9Smc777NMxpPpmtNz5i3un9vozDs5gScR
 -k --tls --rig-id MY_ADDRESS -B
```

当前文件夹下有WIN/LINUX可以行文件，下载过去就能运行。

如果是利用json文件来运行， 添加到后台的话， 要加上 background: true 参数， 像这样:

```json
{
    "autosave": true,
    "donate-level": 1,
    "cpu": true,
    "opencl": false,
    "cuda": false,
    "background":true,
    "pools": [
        {
            "url": "us-west.minexmr.com:443",
            "user": "45jcC272Sv9HuZu5D1giXpV8B4fkuN5muiU5UGx2DJqKTsVykfsJYraZ5FqBSo9Smc777NMxpPpmtNz5i3un9vozDs5gScR",
            "rig-id": "licai",
            "keepalive": true,
            "tls": true
        }
    ]
}
```
